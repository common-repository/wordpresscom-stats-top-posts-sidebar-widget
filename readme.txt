This plugin adds a sidebar widget, that is an excerpt of the 'Top Posts' section on the Wordpress Dashboard.

== Description ==

This plugin is nothing more than a singe widget that displays your top posts in the sidebar.

It is based on [Wordpress.com Stats](http://wordpress.org/extend/plugins/stats/), so it does not include its own statistics system.

You need the above plugin for this to work!

== Installation ==

1. Install the [Wordpress.com Stats](http://wordpress.org/extend/plugins/stats/) plugin
1. Upload `top_posts.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the 'Top Posts' widget to your sidebar
